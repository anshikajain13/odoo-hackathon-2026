# 🚍 TransitOps – Fleet Management System

## Overview

TransitOps is a Fleet Management and Transport Operations Dashboard built using Streamlit and Python.

The system helps transport companies manage:

- Vehicles
- Drivers
- Trips
- Maintenance Records
- Fuel Records
- Cost Analytics

The application provides a centralized dashboard to monitor fleet activities and operational expenses.

---

## Features

### 📊 Dashboard
- Total Vehicles
- Total Drivers
- Active Trips
- Maintenance Records
- Fuel Expenses
- Cost Analytics
- Recent Activity Tracking

### 🚍 Vehicle Management
- Add vehicles
- Track vehicle status
- Search vehicles
- Maintenance status monitoring

### 👨‍✈️ Driver Management
- Add drivers
- Store license information
- Manage contact details

### 🗺️ Trip Management
- Create trips
- Assign vehicles
- Assign drivers
- Monitor trip status

### 🔧 Maintenance Management
- Record maintenance issues
- Track repair costs
- Monitor maintenance history

### ⛽ Fuel Management
- Log fuel entries
- Track fuel expenses
- Monitor vehicle fuel usage
