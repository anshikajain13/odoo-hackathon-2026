Features:
- Authentication & Role-Based Access Control (RBAC)
* Secure login system tailored for distinct operational roles.
* Dedicated dashboard views for Fleet Managers, Drivers, Safety Officers, and Financial Analysts to ensure data privacy and operational focus
- Maintenance & Expense Tracking
* Centralized logging for fuel consumption (liters, cost, date) and ongoing toll/maintenance expenses.
* Dynamic status triggers that automatically switch a vehicle to "In Shop" and hide it from the dispatch pool the moment a maintenance record is generated.
=======
TransitOps is a Fleet Management and Transport Operations Dashboard built using Streamlit and Python.

The system helps transport companies manage:

- Vehicles
- Drivers
- Trips
- Maintenance Records
- Fuel Records
- Cost Analytics

The application provides a centralized dashboard to monitor fleet activities and operational expenses.
